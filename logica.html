Logica de programación

Este es un ejemplo de un ejercicio en java donde el usuario ingresa una contraseña con su documento, si el usuario ingresa corectamente su contraseña podra entrar al menú, en caso de no ser asi, simplemente se le negara el acceso

package com.mycompany.practica;
import java.util.Scanner;
/**
*
* @author Santiago Roman
*/

public class practica {
public static void main(String[] args) {
//Definición de variables
int cedula;
String password;
String passwordDig;

Scanner password1 = new Scanner(System.in);
System.out.println("Cree una contraseña");
password = password1.nextLine();

Scanner cedula1 = new Scanner(System.in);
System.out.println("Ingrese la cedula");
cedula = cedula1.nextInt();

Scanner passwordDig1 = new Scanner(System.in);
System.out.println("Ingrese la contraseña");
passwordDig = passwordDig1.nextLine();

if(password.equals(passwordDig)){
System.out.println("Tienes la opción de ver el siguiente menú");
System.out.println("-Actualizar datos-");
System.out.println("-Ingresar datos-");
System.out.println("-Solicitar cita-");
}else {
System.out.println("Usted no tiene acceso al menú");
}
}
}
